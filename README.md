Agentic, Multimodal Generative AI Fashion Recommendation System

## Overview
StyleSense is an AI-powered virtual fashion stylist that generates
personalized, culturally aware outfit recommendations using
agentic reasoning and multimodal AI.

Unlike traditional fashion apps, StyleSense:
- Works with what users already own
- Respects Indian cultural diversity
- Explains *why* an outfit works
- Educates users through outfit evaluation

## Key Features
- Personalized outfit generation (age, gender, occasion)
- Online wardrobe with image understanding
- Cultural & regional styling modes (India-focused)
- Vintage fashion mode with classic UI
- Outfit evaluation with style scoring and improvements

## AI Architecture
StyleSense uses a **multi-agent AI system**:
1. Wardrobe Vision Agent
2. User Context Agent
3. Cultural Intelligence Agent
4. Outfit Generator Agent
5. Outfit Critic & Evaluator Agent

## Tech Stack
- Groq API (LLM reasoning & generation)
- Hugging Face (Vision-Language Models)
- Serverless APIs
- Low-code frontend
- GitHub for versioning & deployment

## Impact
- Promotes cultural inclusivity
- Reduces fashion waste
- Makes AI explainable and educational
